Opciones para Tarjetas Visa Prepago de Incentivos

Aclaración importante para asesores externos

🔌 Encender / Apagar
Uso Temporal
  • Se apaga la tarjeta de forma momentánea.
  • Útil si se extravía o no la encuentras.
  • Se reactiva al encontrarla.
🔒 Bloquear Tarjeta
Tarjeta Perdida / Robo
  • El plástico se bloquea y se genera una nueva tarjeta.
  • Caso de pérdida o robo.
  • Se emite un nuevo plástico.
❌ Inactivar Tarjeta
Fin de Vinculación
  • Tarjeta inhabilitada permanentemente.
  • Finaliza el vínculo laboral.
  • No se consignarán más incentivos.

¿Qué solicitudes puedes realizar?

Gestión disponible a través del área de incentivos

01 🔐 Bloqueo de Tarjeta
  • Puedes solicitar el bloqueo desde la plataforma.
  • También puedes comunicarte directamente con el banco.

Una vez realizado el bloqueo, el plástico anterior se invalida y se genera una nueva tarjeta.

02 🔑 Solicitud de PIN
  • La solicitud debe realizarse a través de esta área.
  • Desde aquí se gestiona la generación del PIN para la activación de la tarjeta.
  • Esto generará un nuevo plástico con el PIN que será entregado posteriormente.
03 📝 Actualización o Modificación de Datos
  • En caso de requerir cambios en los datos registrados en la plataforma Bancolombia, es necesario enviar la solicitud con la información completa para poder realizar la actualización.
